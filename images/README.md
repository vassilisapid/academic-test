# Image assets

Copy these files from the old `vassilisapid.github.io` repository into this directory:

- `photoVassi.jpg` — used as the Academic Pages profile/avatar image.
- `ArchimedesIm.png` — optional Archimedes logo, if you want to reuse it.

The `_config.yml` file expects the profile image at:

```text
images/photoVassi.jpg
```
